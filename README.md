# ANALISI FABBISOGNI

Dati i codici univoci di articoli e le relative quantità si vuole conoscere la quantità di materie prime da impiegare per produrre le quantità specificate di quegli articoli.

## Come funziona

1. Caricare il file CSV con i dati delle distinte base
2. Inserire i codici univoci degli articoli e le relative quantità
3. Cliccare su "Analizza"
4. Scaricare il file Excel con i risultati
